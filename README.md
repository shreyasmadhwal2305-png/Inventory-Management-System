# Inventory Management System (C)

## Description
A file-based inventory system built in C that allows users to manage products efficiently using CRUD operations.

## Features
- Add Product
- View Products
- Search Product
- Update Product
- Delete Product
- Count Total Products
- Clear All Records

## Concepts Used
- File Handling (fread, fwrite)
- Structures
- CRUD Operations
- Input Validation

## How to Run

gcc main.c -o inventory  
./inventory
